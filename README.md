# Лендинг для Ольги (AI‑дизайнер)

## Как открыть

- Откройте файл `index.html` в браузере (двойной клик).

## Что настроить

- В `index.html` замените ссылку `https://t.me/` на вашу реальную ссылку на портфолио/контакт в Telegram (она встречается в нескольких местах).

## Файлы

- `index.html` — разметка
- `styles.css` — стили + светлая/тёмная темы
- `script.js` — переключатель темы + год в футере
